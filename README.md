# Location-Finder
A simple library to find the current location of the device

How to use

Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
 Add the dependency

	dependencies {
	        implementation 'com.github.Kavya-24:Location-Finder:Tag'
	}
